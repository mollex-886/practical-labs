Investigate the provided files and analyze suspicious activity related to the XMLRat malware.

Reviewed the provided files and checked for suspicious artifacts, unusual file names, and XML-related content.

- Reviewing file contents

## Step 2 - Network Traffic Analysis
Opened the provided packet capture (PCAP) file in Wireshark to inspect network activity.

Focused on:
- Suspicious IP addresses
- Unusual traffic patterns
- Potential command-and-control communication
- DNS and HTTP requests

Tools used:
- Wireshark

---

## Step 3 - Malware Investigation
Analyzed suspicious indicators and investigated possible malware behavior.

Focused on:

Tools used:
- VirusTotal
